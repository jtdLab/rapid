# Ui Command

Work with the UI part of an existing Rapid project.

```
rapid ui <platform>
```

<TabItem value="android">
- `rapid ui android`: Work with the Android UI part of an existing Rapid project.
  - `rapid ui android add`: Add components to the Android UI part of an existing Rapid project.
    - `rapid ui android add widget`: Add a widget to the Android UI part of an existing Rapid project.
  - `rapid ui android remove`: Remove components from the Android UI part of an existing Rapid project.
    - `rapid ui android remove widget`: Remove a widget from the Android UI part of an existing Rapid project.
</TabItem>
