# Infrastructure Command

Work with the infrastructure part of an existing Rapid project.

```
rapid infrastructure <subcommand>
```

## Subcommands

- `rapid infrastructure <subinfrastructure>`: Work with a subinfrastructure.
  - `rapid infrastructure <subinfrastructure> add`: Add a component to a subinfrastructure.
    - `rapid infrastructure <subinfrastructure> add data_transfer_object`: Add a data transfer object to a subinfrastructure.
    - `rapid infrastructure <subinfrastructure> add service_implementation`: Add a service implementation to a subinfrastructure.
  - `rapid infrastructure <subinfrastructure> remove`: Remove a component from a subinfrastructure.
    - `rapid infrastructure <subinfrastructure> remove data_transfer_object`: Remove a data transfer object from a subinfrastructure.
    - `rapid infrastructure <subinfrastructure> remove service_implementation`: Remove a service implementation from a subinfrastructure.
