Work with the domain part of an existing Rapid project.

```

```

## Subcommands

- `rapid domain add`: Add subdomains to the domain part of an existing Rapid project.
  - `rapid domain add sub_domain`: Add subdomains of the domain part of an existing Rapid project.
- `rapid domain <subdomain>`: Work with a subdomain.
  - `rapid domain <subdomain> add`: Add a component to a subdomain.
    - `rapid domain <subdomain> add entity`: Add a entity to a subdomain.
    - `rapid domain <subdomain> add service_interface`: Add a service interface to a subdomain.
    - `rapid domain <subdomain> add value_object`: Add a value object to a subdomain.
  - `rapid domain <subdomain> remove`: Remove a component from a subdomain.
    - `rapid domain <subdomain> remove entity`: Remove a entity from a subdomain.
    - `rapid domain <subdomain> remove service_interface`: Remove a service interface from a subdomain.
    - `rapid domain <subdomain> remove value_object`: Remove a value object from a subdomain.
- `rapid domain remove`: Remove subdomains from the domain part of an existing Rapid project.
  - `rapid domain remove sub_domain`: Remove subdomains from the domain part of an existing Rapid project.

## `rapid domain add sub_domain`

```
rapid domain add sub_domain <name>
```

## `rapid domain <subdomain> add entity`

```
rapid domain <subdomain> add entity <name> [arguments]
```
