# Infrastructure Command

Work with the infrastructure part of an existing Rapid project.

```
rapid infrastructure <subcommand>
```

## Subcommands

- `rapid infrastructure <subinfrastructure>`: Work with the subinfrastructure <subinfrastructure>.
  - `rapid infrastructure <subinfrastructure> add`: Add a component to the subinfrastructure 

## `rapid infrastructure <subinfrastructure> add data_transfer_object`

```
rapid infrastructure <subinfrastructure> add data_transfer_object [arguments]
```
