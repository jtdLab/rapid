# Activate

This command adds support for a platform to the current project.

```bash
Add support for a platform to an existing Rapid project.

Usage: rapid activate <platform>
-h, --help    Print this usage information.

Available subcommands:
  android   Adds support for Android to this project.
  ios       Adds support for iOS to this project.
  linux     Adds support for Linux to this project.
  macos     Adds support for macOS to this project.
  mobile    Adds support for Mobile to this project.
  web       Adds support for Web to this project.
  windows   Adds support for Windows to this project.

Run "rapid help" to see global options.
```
